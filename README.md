
# devcraft_studio

Интерактивный лендинг для веб-студии DevCraft - разработка сайтов, мобильных приложений и CRM/ERP систем.

## Технологии

- React + TypeScript
- Vite
- Lucide React (иконки)
- Radix UI (компоненты)

## Установка и запуск

Установите зависимости:
```bash
npm i
```

Запустите dev-сервер:
```bash
npm run dev
```

Создайте production-сборку:
```bash
npm run build
```
